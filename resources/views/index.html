Bersama Membangun Masa Depan

Himpunan Mahasiswa yang berkomitmen untuk mengembangkan potensi mahasiswa melalui berbagai kegiatan akademik, sosial, dan pengembangan diri.

Tentang Kami

Inovasi

Mengembangkan ide-ide kreatif dan inovatif untuk kemajuan mahasiswa dan universitas.

Kolaborasi

Membangun kerjasama yang solid antar mahasiswa untuk mencapai tujuan bersama.

Prestasi

Mendorong setiap mahasiswa untuk meraih prestasi terbaik di berbagai bidang.

Program Kerja

Program Akademik

Seminar, workshop, dan pelatihan untuk meningkatkan kompetensi akademik mahasiswa.

Program Sosial

Kegiatan bakti sosial dan pengabdian masyarakat untuk membangun kepedulian sosial.
